LLM-BASED VIDEO TO BLOG GENERATOR

The LLM-Based Video to Blog Generator is an AI-powered web application designed to automatically convert YouTube videos into well-structured, SEO-friendly blog posts. The system extracts video transcripts and utilizes Large Language Models (LLMs) to generate high-quality written content, eliminating the need for manual transcription and rewriting.

By integrating transcript extraction, prompt engineering, and generative AI, the platform streamlines content repurposing for creators, educators, and marketers. This solution improves productivity, reduces turnaround time, and enables consistent content creation across digital platforms using modern AI technologies.

Technologies Used

Backend: Node.js, Express.js
AI Model: Google Gemini Pro (LLM)
APIs: YouTube Transcript API
Frontend: HTML, CSS, JavaScript
